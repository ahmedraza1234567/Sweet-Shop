# Sweet Shop 🍭

This is a Django-based online sweet shop web application where users can view and order sweets.

## 🚀 Features:
- Admin Login
- User Registration & Login
- Sweets Management (Add, Edit, Delete)
- Place Orders
- Order History

## 🛠 Tech Stack:
- Python (Django Framework)
- HTML, CSS, Bootstrap
- SQLite Database
